2. mapToInt() method!
mapToInt() converts the stream to a primitive stream,
which avoids boxing overhead and provides numeric operations like sum, max, and average directly.
| Feature     | map()           | mapToInt() |
| ----------- | --------------- | ---------- |
| Output      | Stream<Integer> | IntStream  |
| Boxing      | Yes             | No         |
| Performance | Slower          | Faster     |
| Numeric ops | Limited         | Rich       |
